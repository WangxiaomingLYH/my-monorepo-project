# 数据类型及数学运算符
1. int: 整数
2. float: 浮点数
3. str: 字符串
4. bool: 布尔值
5. % 余数; ** 根数, 如 a ** 2即a的平方根

# 基础
1. 定义变量无需使用关键字: ` name='lyh' `
2. 输出使用 print: print(name)  ===> lyh
3. 使用 input 获取用户输入: ` username = input('Enter your name:') `, 默认存储为字符串
4. 使用 int() 将用户输入存储为数值: ` int(input('What is your age?')) `
